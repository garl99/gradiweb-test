# Gradiweb Test

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Technologies

* VueJS 2
* HTML5
* SASS
* JavaScript

### Features

* Responsive Design
* Data and documentation provided by [OpenWeather](https://openweathermap.org/)
